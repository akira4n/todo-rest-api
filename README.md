Sebuah project RESTful API TO-DO List sederhana yang dibangun menggunakan Laravel 12.

| Method | Endpoint    | Deskripsi                |
| ------ | ----------- | ------------------------ |
| GET    | /tasks      | Ambil semua daftar to-do |
| POST   | /tasks      | Buat to-do baru          |
| GET    | /tasks/{id} | Ambil detail satu to-do  |
| PUT    | /tasks/{id} | Update to-do             |
| DELETE | /tasks/{id} | Hapus to-do              |
